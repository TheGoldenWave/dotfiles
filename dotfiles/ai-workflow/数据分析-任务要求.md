# 数据分析 - 任务要求

> 每周五 11:00 执行。产出一份数据草稿供团队会议使用。
> 修改此文件后，下周五自动生效，无需改 cron 配置。

## 数据来源

### 1. 知识库变更
- `cd /Users/goldenwave/KnowledgeBase`
- 运行 `git log --since="last Friday" --oneline --all`
- 提取本周新增/修改的文件清单

### 2. inbox 扫描
- 扫描 `/Users/goldenwave/KnowledgeBase/inbox/` 下本周修改的文件
- 列出文件名和简要主题

### 3. Multica Issue 统计
- 使用 `~/bin/multica` 获取各 workspace 本周完成的 Issue
- Workspace 列表：
  - AI先锋团 (f1720aa2-901a-4889-809a-9ffd498783e9)
  - 直播课AI&数据 (b2b717cd-0ae1-4a1e-97f4-38a06eacf47a)
  - GoldenWave (a2634170-fac8-4f20-9541-4c4eb2dfb04f)

### 4. 项目进展（如有）
- 检查 `~/Documents/MyProject/Zhiboke_Claw/` 是否有本周关键变更
- 检查 `~/Documents/MyProject/goldenwave-asia/` 是否有本周关键变更
